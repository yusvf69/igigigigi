## 1. EXECUTIVE OVERVIEW

### 1.1 Purpose

Social Learning connects learners in a supportive community where they can study together, discuss ideas, ask questions, and help each other grow in Islamic knowledge — all within a safe, moderated environment that upholds Islamic adab.

### 1.2 Core Principle

**Learning is enhanced through community. Knowledge shared is knowledge multiplied.**

---

## 2. SOCIAL LEARNING PHILOSOPHY

### 2.1 Islamic Community Learning Tradition

| Tradition | Description | Digital Implementation |
|---|---|---|
| **Halaqah** | Study circle | Study groups with structured curriculum |
| **Talab al-Ilm** | Seeking knowledge together | Collaborative learning paths |
| **Mudhakarah** | Reviewing knowledge together | Peer quizzing and review |
| **Ijazah** | Permission to teach | Mentorship and teaching badges |
| **Naseehah** | Sincere advice | Constructive feedback culture |

### 2.2 Social Learning Principles

| Principle | Description |
|---|---|
| **Adab First** | All interactions follow Islamic etiquette |
| **Beneficial Speech** | Only speak what is beneficial or remain silent |
| **Respectful Disagreement** | Differences of opinion expressed with respect |
| **No Backbiting** | Strict prohibition of gheebah |
| **Encouragement** | Uplift and support fellow learners |
| **Knowledge with Humility** | Share knowledge without arrogance |

## 3. COMMUNITY STRUCTURE

### 3.1 Community Levels

| Level | Scope | Moderation | Purpose |
|---|---|---|---|
| **Global Community** | All platform users | Platform moderators | General discussions, announcements |
| **Topic Hubs** | Per knowledge domain | Topic experts | Focused discussions on a domain |
| **Study Groups** | Small groups (5-20) | Group admin | Collaborative learning |
| **Classrooms** | Educator + students | Educator | Structured education |
| **Friend Circles** | User-defined | Self-moderated | Private sharing |

### 3.2 User Roles

| Role | Permissions | Responsibilities |
|---|---|---|
| **Learner** | Participate, post, comment | Follow community guidelines |
| **Study Group Admin** | Manage group, approve members | Guide group learning |
| **Educator** | Create classrooms, assign content | Teach and assess |
| **Moderator** | Moderate all content | Enforce guidelines |
| **Scholar** | Verified Islamic scholar | Provide authoritative guidance |
| **Admin** | Platform administration | Full platform oversight |

## 6. PEER LEARNING

### 6.1 Peer Learning Features

| Feature | Description |
|---|---|
| **Peer Quizzing** | Learners quiz each other on topics |
| **Flashcard Sharing** | Share decks with study groups |
| **Explanation Requests** | Ask a peer to explain a concept |
| **Study Buddy Matching** | Algorithm matches learners at similar levels |
| **Peer Review** | Review explanations written by peers |
| **Mentorship** | Advanced learners mentor beginners |

### 6.2 Study Buddy Matching

| Criteria | Weight |
|---|---|
| **Knowledge Level** | 0.30 | Same level or one level apart |
| **Active Topics** | 0.25 | Overlapping topics of study |
| **Language** | 0.20 | Same primary language |
| **Time Zone** | 0.15 | Within 3 hours |
| **Learning Style** | 0.10 | Complementary styles |

## 8. MODERATION AND SAFETY

### 8.1 Moderation Levels

| Level | Method | Response Time |
|---|---|---|
| **Automated** | AI filter for profanity, hate speech, spam | Real-time |
| **Community** | User reporting system | < 1 hour |
| **Moderator** | Human moderator review | < 4 hours |
| **Scholar** | Islamic content review by verified scholars | < 24 hours |

### 8.2 Community Guidelines

| Rule | Description | Enforcement |
|---|---|---|
| **Respectful Speech** | No insults, mockery, or condescension | Warning → suspension |
| **No Backbiting** | No gossip about individuals | Warning → suspension |
| **Stay On Topic** | Posts relevant to the category | Post removal |
| **No Spam** | No self-promotion or repetitive posts | Ban |
| **Cite Sources** | Factual claims should reference sources | Reminder → removal |
| **No Fatwa Shopping** | Seeking multiple opinions to find easy answers | Redirect to scholars |

### 8.3 Reporting System

| Feature | Description |
|---|---|
| **Report Types** | Spam, harassment, misinformation, inappropriate content |
| **Anonymous Reporting** | Report without revealing identity |
| **Appeal Process** | Appeal moderation decisions |
| **Moderation Transparency** | Public moderation log (anonymized) |
