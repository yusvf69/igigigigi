## 1. EXECUTIVE OVERVIEW

### 1.1 Purpose

The Gamification Architecture provides motivational mechanics that encourage consistent learning, reward achievement, and foster healthy competition — all while respecting Islamic values. It transforms the learning journey into an engaging experience without trivializing sacred knowledge.

### 1.2 Core Principle

**Gamification serves learning, not distraction. Rewards recognize knowledge gained, not just time spent.**

---

## 2. GAMIFICATION PHILOSOPHY

### 2.1 Islamic Framework for Gamification

| Principle | Description |
|---|---|
| **Niyyah (Intention)** | Mechanics encourage learning for the sake of knowledge, not just rewards |
| **Tawadhu (Humility)** | Leaderboards emphasize progress over ranking |
| **Ihsan (Excellence)** | Rewards quality and depth, not just quantity |
| **Adl (Justice)** | Fair systems that don't disadvantage new or slow learners |
| **Taqwa (Consciousness)** | No mechanics that encourage showing off (riya) |
| **Barakah (Blessing)** | Spiritual rewards (hasanat) recognized alongside gamified ones |

### 2.2 Motivation Types

| Type | Description | Game Element |
|---|---|---|
| **Intrinsic** | Love of learning, curiosity | Content quality, discovery |
| **Achievement** | Sense of accomplishment | Badges, levels, trophies |
| **Social** | Recognition and community | Leaderboards, sharing |
| **Progress** | Visible advancement | XP, progress bars |
| **Consistency** | Building habits | Streaks, daily goals |
| **Mastery** | Deep understanding | Mastery badges, challenges |

## 4. POINTS AND XP SYSTEM

### 4.1 XP Sources

| Activity | Base XP | Bonus Factors |
|---|---|---|
| **Complete a flashcard review** | 5 XP per card | Streak multiplier, difficulty |
| **Pass a quiz** | 10 XP per question (correct) | Difficulty multiplier, streak |
| **Read a story** | 50 XP | Completion, comprehension check |
| **Listen to a podcast** | 40 XP | Full listening, comprehension check |
| **AI Teacher interaction** | 2 XP per turn | Quality of interaction |
| **Daily login** | 10 XP | Streak bonus |
| **Complete a challenge** | 100-500 XP | Difficulty |
| **Help others (social)** | 20 XP | Peer recognition |
| **Teaching/reviewing** | 30 XP | Quality of review |

### 4.2 XP Multipliers

| Multiplier | Condition | Effect |
|---|---|---|
| **Streak Bonus** | Consecutive days active | +10% per day (max 100%) |
| **Difficulty Bonus** | Hard/Expert content | x1.5 for Hard, x2 for Expert |
| **First Attempt Bonus** | First time answering correctly | +25% |
| **Perfect Score** | 100% on a quiz | +50% |
| **Consistency Bonus** | Same time daily | +10% |
| **Ramadan Multiplier** | During Ramadan | x2 |

## 7. LEADERBOARDS

### 7.1 Leaderboard Types

| Type | Scope | Reset | Opt-Out |
|---|---|---|---|
| **Daily** | All learners | Daily | Yes |
| **Weekly** | All learners | Weekly | Yes |
| **Monthly** | All learners | Monthly | Yes |
| **Topic** | Per knowledge domain | Never | Yes |
| **Friends** | User's friend group | Weekly | Yes |
| **Study Group** | Group members | Per challenge | No |

### 7.2 Islamic Leaderboard Design

| Feature | Implementation |
|---|---|
| **Anonymous by Default** | Usernames, not real names |
| **No Negative Display** | Show top learners, never show "last place" |
| **Group-Based Optional** | Default off, opt-in required |
| **Focus on Progress** | "Most improved" alongside top scores |
| **Humility Reminders** | Periodic notifications: "Let your left hand not know what your right hand gives" |
| **No Late-Night Competition** | Leaderboard frozen during sleep hours |

## 8. STREAKS AND CONSISTENCY

### 8.1 Streak Mechanics

| Element | Description |
|---|---|
| **Streak Definition** | Complete at least one learning activity per calendar day |
| **Streak Tracking** | Consecutive days counter |
| **Freeze Option** | 1 freeze per week (miss a day without losing streak) |
| **Grace Period** | 6-hour grace into next day (e.g., after Isha) |
| **Streak Recovery** | If broken, 3-day grace period to restore with double XP |

### 8.2 Streak Rewards

| Streak Length | Reward |
|---|---|
| 7 days | 7-Day Streak Badge + 100 XP bonus |
| 30 days | 30-Day Streak Badge + special profile frame |
| 100 days | 100-Day Streak Badge + exclusive content unlock |
| 365 days | Year of Knowledge Badge + personalized du'a from scholars |

## 9. CHALLENGES AND QUESTS

### 9.1 Challenge Types

| Type | Duration | Description |
|---|---|---|
| **Daily Challenge** | 1 day | 3-5 questions on random topic |
| **Weekly Quest** | 7 days | Complete a learning path (e.g., "Tawheed Week") |
| **Topic Mastery** | Variable | Complete all content in a topic at 90%+ accuracy |
| **Ramadan Challenge** | 30 days | Special Ramadan learning goals |
| **Community Challenge** | Variable | Group goals (e.g., "1,000 verses memorized by the group") |

### 9.2 Quest Structure

| Stage | Description | Reward |
|---|---|---|
| **Quest Start** | Quest briefing with objectives | — |
| **Task 1** | Easy introductory task | Small XP |
| **Task 2** | Medium knowledge task | Medium XP |
| **Task 3** | Hard challenge task | Large XP |
| **Boss Task** | Comprehensive assessment | Major XP + badge |
| **Completion** | Quest complete celebration | Title unlock + currency |
