## 1. EXECUTIVE SUMMARY

### 1.1 Purpose

This document defines every responsibility in the AI company using a RACI (Responsible, Accountable, Consulted, Informed) framework. Every task, decision, and deliverable has a clearly assigned owner.

### 1.2 Core Principle

**Every responsibility has exactly one accountable party. Shared responsibility means no responsibility.**

---

## 2. RESPONSIBILITY FRAMEWORK

### 2.1 RACI Definitions

| Code | Meaning | Description |
|---|---|---|
| **R** | Responsible | Does the work |
| **A** | Accountable | Answers for the outcome (only one per task) |
| **C** | Consulted | Provides input before decision |
| **I** | Informed | Notified after decision |

### 2.2 Responsibility Levels

| Level | Scope | Examples |
|---|---|---|
| **Strategic** | Company-wide | Define knowledge strategy |
| **Tactical** | Department-wide | Design verification workflow |
| **Operational** | Team-level | Execute OCR on page |
| **Execution** | Task-level | Generate quiz question |
